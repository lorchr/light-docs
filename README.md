# Dependency
```shell
node.js > 6.x
```

# Usage
## Install
```shell
npm install -g docsite
```

## Init a project

```shell
$ cd my-project
$ docsite init
# or
$ docsite init my-project
```

## Local preview
```shell
$ docsite start
```

## Build
```shell
$ docsite build
```

## 访问连接

- http://localhost:8080
- https://lorchr.github.io/light-docs
